# MJ AI Tools - Internal Tools

Internal tools used by other binaries. These are not intended for direct user interaction.

## Tools Overview

### aicont-runner

AiCont Runner is a task executor that watches for task definition files and executes them using aicmd.

[Full documentation](./aicont-runner.md)

## Usage

Tools are available in the `stai-tools/bin-inc/` directory and support:

- `--version` - Show version information
- `--help` - Show usage information
- `--readme` - Show full documentation
